# 自选股综合 Dashboard

## 项目简介

股票智能看板 Dashboard，覆盖 A股、港股、美股数据。用户可以自行添加自选股，对自选股进行技术分析，并通过东方财富、同花顺、雪球、新浪财经等主流财经网站入口，结合市场新闻、公司动态、讨论区情绪面等信息，多角度了解自选股动态。

本项目可作为股票投资者了解持仓和自选股基本情况的基础工具，支持本地部署，也可部署为公网网页。页面提供自选股输入提示、示例组合和分析入口，欢迎试用。

> 说明：当前版本以行情抓取、技术分析、持仓财富管理、板块概念映射、新闻入口和财经网站入口为主。所有持仓成本、数量和备注默认保存在用户浏览器本地。

## 主要功能

- 支持 A股、港股、美股混合自选股输入。
- 自动按 A股、港股、美股分组展示。
- 展示实时行情、涨跌幅、20日均线距离、量能比等指标。
- 根据均线、RSI、量能、支撑位、压力位等指标生成趋势评分。
- 展示关键点位，包括支撑位、压力位、止损参考、止盈一档、止盈二档和盈亏比。
- 展示日K线、成交量柱和未来趋势虚拟画线预测。
- 输出技术风险提示、成交量分析和潜在催化观察点。
- 每只股票支持填写持仓成本、持股数量和持仓备注，数据保存在浏览器本地。
- 右侧提供财富管理面板，按 A股、港股、美股汇总持仓市值、总成本、总盈亏和总体收益率。
- 根据组合集中度和总体收益率生成一行财富管理建议。
- 展示个股所属板块、相关概念和美股映射参照。
- 提炼个股新闻和板块新闻的关键消息，以标签、标题和摘要方式呈现。
- 提供个股新闻、板块新闻、板块排行入口。
- 每只股票提供东方财富、同花顺、雪球、新浪财经入口。
- 支持本地运行，也支持部署为公网网页。

## 页面使用方式

打开页面后，按以下流程使用：

1. 在顶部“自选股录入”中选择市场：A股、港股或美股。
2. 输入股票代码，并可同步填写持股成本和持股数量。
3. 点击“加入并分析”按钮。
4. 页面会自动按 A股、港股、美股分组展示结果。
5. 查看每只股票的行情、趋势评分、风险提示和催化提示。
6. 在每只股票卡片右侧填写“成本”和“数量”，右侧财富管理面板会自动汇总整体盈亏。
7. 也可以在右侧“添加持仓”中录入市场、股票代码、持股成本和持股数量，系统会自动加入自选并刷新分析。
8. 在“持仓备注”中记录仓位计划、加减仓条件或观察点。
9. 查看关键点位、日K线、成交量、趋势虚拟线、板块概念、美股映射和新闻关键消息。
10. 点击东方财富、同花顺、雪球、新浪财经、个股新闻、板块新闻、板块排行按钮，跳转到外部财经页面查看更详细信息。

页面顶部也提供“示例组合”按钮，点击后会自动填入示例股票并执行分析。

## 股票代码输入格式

支持以下常见写法：

| 市场 | 示例 | 说明 |
|---|---|---|
| A股 | `300620` | 自动识别深市或沪市 |
| A股 | `SZ300620` | 明确指定深市 |
| A股 | `SH600519` | 明确指定沪市 |
| 港股 | `hk00700` | 腾讯控股 |
| 港股 | `00700.HK` | Yahoo 风格写法 |
| 港股 | `700` | 会自动补齐为 `HK00700` |
| 美股 | `AAPL` | 苹果 |
| 美股 | `TSLA` | 特斯拉 |

一次输入多个股票的示例：

```text
300620 300394 hk00700 AAPL TSLA
```

也可以换行输入：

```text
300620
300394
hk00700
AAPL
TSLA
```

## 本地运行

适合自己电脑上使用。

### 环境要求

- Python 3.10 或更高版本
- 能访问腾讯行情接口和 Yahoo Chart 接口

### 运行步骤

```powershell
git clone https://github.com/hanszeng626-blip/watchlist-dashboard.git
cd watchlist-dashboard
pip install -r requirements.txt
python app.py
```

启动成功后，终端会显示：

```text
Watchlist dashboard: http://127.0.0.1:8765
```

浏览器打开：

```text
http://127.0.0.1:8765
```

也可以在 URL 中直接带入自选股：

```text
http://127.0.0.1:8765/?symbols=300620%20hk00700%20AAPL
```

Windows 用户也可以双击 `start_dashboard.bat`，脚本会自动启动本地服务并打开 Dashboard 页面。

## 公网部署

适合让其他人直接通过网址访问，不需要他们在本地安装 Python。

本项目已经支持云平台部署。部署平台需要能运行 Python Web Service，并提供公网访问域名。

### Render 部署方式

1. 打开 [Render](https://render.com/)。
2. 注册或登录账号。
3. 选择 `New` -> `Web Service`。
4. 连接本 GitHub 仓库：`hanszeng626-blip/watchlist-dashboard`。
5. Build Command 填：

```bash
pip install -r requirements.txt
```

6. Start Command 填：

```bash
python app.py
```

7. Environment Variable 增加：

```text
HOST=0.0.0.0
```

8. 点击部署。
9. 部署完成后，Render 会生成一个公网 URL，其他人打开这个 URL 即可使用。

仓库中也提供了 `render.yaml`，支持按 Render Blueprint 方式创建服务。

## 数据来源

- 实时报价：腾讯行情接口。
- A股/港股日线：腾讯 K 线接口。
- 美股日线及兜底日线：Yahoo Chart 公开接口。
- 财经网站入口：东方财富、同花顺、雪球、新浪财经。
- 新闻入口：Bing News RSS 和财经网站搜索入口。
- 新闻关键消息：基于公开 RSS 标题和摘要做轻量提炼。
- 板块/概念：基于股票名称和代码做轻量规则映射，作为投资研究入口，不作为严格行业分类。

## 注意事项

- 本工具用于行情观察和技术分析辅助，不构成投资建议。
- 行情、技术指标、止盈止损、趋势虚拟线、板块映射和财富管理建议仅供参考，应结合基本面、仓位、风险承受能力和市场环境综合判断。
- 如果部署平台无法访问外部行情接口，页面可能无法正常返回行情数据。
- 持仓成本、持股数量和备注保存在浏览器本地，更换浏览器或清除浏览器数据后可能丢失。
